# flutter_med_anvisa

Flutter app that reads the barcode of a drug and shows some information about it.

ANVISA Drug List (Brazil) updated on 05/30/2020.

## Packages
- flutter_barcode_scanner: ^1.0.1
- get: ^2.10.3
- font_awesome_flutter: ^8.8.1
